# cosmic-on-air
http://cosmic-on-air.org web page. 
Cosmic On Air is a citizen science project led by IRSN, OBSPM, PLANETE SCIENCES, MNHN.
Help scientists and advance science by measuring cosmic radiation on board an aircraft.
